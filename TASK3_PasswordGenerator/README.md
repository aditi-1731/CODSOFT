# Password Generator

## Description
A command-line based Password Generator developed in Python. The application generates strong and random passwords based on user-defined length and selected character types.

This project was developed as part of the **CodSoft Python Programming Internship**.

## Features
- Generate secure random passwords
- User-defined password length
- Include uppercase letters
- Include lowercase letters
- Include numbers
- Include special characters
- Input validation
- Generate multiple passwords in one session

## Technologies Used
- Python 3
- `random` module
- `string` module
- Functions
- Input Validation

## How to Run

```bash
python password_generator.py
```
