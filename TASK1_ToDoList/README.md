# To-Do List Application

## Description

A command-line based To-Do List Manager developed in Python. The application allows users to add, view, complete, and delete tasks while storing data persistently using file handling.

## Features

* Add new tasks
* View all tasks
* Mark tasks as completed
* Delete tasks
* Store tasks in a text file
* Load saved tasks automatically on startup

## Technologies Used

* Python 3
* File Handling
* Functions
* Exception Handling

## How to Run

```bash
python todo.py
```
