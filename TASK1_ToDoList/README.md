# To-Do List Application

## Description

A command-line based To-Do List Manager developed in Python. The application helps users manage and organize daily tasks efficiently. Tasks are stored persistently using file handling, allowing users to access their saved tasks even after restarting the program.

## Features

* Add new tasks
* View all tasks
* Update existing tasks
* Mark tasks as completed
* Delete tasks
* View task statistics (Total, Completed, Pending)
* Input validation for better user experience
* Store tasks in a text file
* Automatically load saved tasks on startup

## Technologies Used

* Python 3
* File Handling
* Functions
* Lists
* Exception Handling

## How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/aditi-1731/CODSOFT.git
   ```

2. Navigate to the project folder:

   ```bash
   cd CODSOFT/TASK1_ToDoList
   ```

3. Run the application:

   ```bash
   python todo.py
   ```
