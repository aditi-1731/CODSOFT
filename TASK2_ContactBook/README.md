# Contact Book Application

## Description

The Contact Book Application is a command-line based Python project that helps users manage their contacts efficiently. The application allows users to add, view, search, update, and delete contact information while storing data persistently using file handling.

This project was developed as part of the CodSoft Python Programming Internship.

---

## Features

* Add new contacts
* View all saved contacts
* Search contacts by name
* Update existing contact details
* Delete contacts
* Validate phone numbers
* View contact statistics
* Store contacts permanently using file handling
* Automatically load saved contacts on startup

---

* Python 3
* Dictionaries
* Functions
* File Handling

## How to Run

1. Make sure Python 3 is installed on your system.
2. Open a terminal in the project directory.
3. Run the following command:

```bash
python contact_book.py
```
