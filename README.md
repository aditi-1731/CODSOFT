# CODSOFT Python Programming Internship

This repository contains the Python projects completed as part of my **Python Programming Internship at CodSoft**. Each project demonstrates fundamental Python programming concepts, including file handling, functions, data structures, input validation, exception handling, and modular programming.

## Projects

### 📌 Task 1: To-Do List Application
A command-line application to manage daily tasks.

**Features:**
- Add tasks
- View tasks
- Update tasks
- Mark tasks as completed
- Delete tasks
- View task statistics
- Persistent storage using file handling

---

### 📌 Task 2: Contact Book Application
A command-line contact management system.

**Features:**
- Add contacts
- View contacts
- Search contacts
- Update contact details
- Delete contacts
- Phone number validation
- Store contact name, phone number, email, and address
- Contact statistics
- Persistent storage using file handling

---

### 📌 Task 3: Password Generator
A secure password generator with customizable options.

**Features:**
- User-defined password length
- Include uppercase letters
- Include lowercase letters
- Include numbers
- Include special characters
- Input validation
- Generate multiple passwords in one session
